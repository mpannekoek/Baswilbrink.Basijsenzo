# Technology Stack

## Programming Languages
- TypeScript - configured via `src/web/tsconfig.json` - application code and tests
- CSS - `src/web/app/globals.css` - styling

## Frameworks
- Next.js 16.2.1 - routing, server rendering, route handlers, standalone output
- React 19.2.4 - component runtime
- NextAuth 4.24.13 - admin authentication and session handling

## Infrastructure
- Standalone Node.js web application output
- Azure AD authentication provider for admin login

## Build Tools
- npm - script runner and package management
- ESLint 9.39.4 - linting
- Tailwind CSS 4.2.2 - styling pipeline

## Testing Tools
- Vitest 4.1.2 - unit/component test runner
- Testing Library - component rendering and DOM assertions
- jsdom - browser-like environment for tests
